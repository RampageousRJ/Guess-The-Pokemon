# Guess-The-Pokemon

## Project Description

Guess-The-Pokemon is a web-based interactive game that challenges users to identify Pokémon characters. The game displays the image of a Pokémon as an additional hint along with other details, and the user must input their guess. It is a fun and engaging way to test your Pokémon knowledge.

## Installation Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/RampageousRJ/Guess-The-Pokemon.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Guess-The-Pokemon
   ```

3. **Open the `index.html` file in a web browser to start the game.**

No additional installations or dependencies are required.

## Usage

- Open `index.html` in your preferred web browser.
- Input your guess in the text box provided and submit to see if you can guess the Pokémon correctly.
- Feedback will be given immediately to inform you if your guess is correct.

## Features

- Simple and intuitive user interface.
- Instant feedback on your guesses.
- Endless gameplay with a wide variety of Pokémon silhouettes.
